# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Scope

Applies to all project spaces: repository, issue tracker, pull requests, discussions, chat, and any event where the project is represented.

## Reporting

Concerns may be reported to the maintainers via the channel listed in `SECURITY.md`. Reports are handled confidentially.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate the Contributor Covenant. Repeated or severe violations may result in temporary or permanent exclusion from project spaces.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
