# Contextual Districts (A, B, D, X Suffixes)

Contextual zoning districts replace the open-ended height-factor system with mandatory height limits and streetwall requirements. They produce predictable building forms that relate to the existing neighborhood character.

## Quality Housing Program

All contextual districts use the **Quality Housing** program (ZR Article II, Chapter 8). Quality Housing is **mandatory** in contextual districts (optional in standard R6–R10).

Key Quality Housing requirements:
- Mandatory maximum building height
- Required base height range (the streetwall must rise to at least the minimum and no more than the maximum)
- Required setback above the base
- Planting requirements (street trees, rear yard landscaping)
- Recreation space for buildings with 10+ units
- Refuse storage rooms
- Laundry rooms or washer/dryer hookups
- Bicycle storage

## Suffix Meanings

| Suffix | Character | Typical Form |
|--------|-----------|--------------|
| **A** | Lower height, wider streetwall | 6–8 story contextual buildings, common in brownstone/rowhouse areas |
| **B** | Taller, tower-on-base allowed | 10–20+ story towers above a required base, common on wide streets |
| **D** | Narrow lot variant | Taller/slimmer than A, for lots ≤45 ft wide (infill on narrow frontages) |
| **X** | Hybrid | Contextual base with some height-factor flexibility above |

## How to Apply Contextual Rules

1. **Identify the suffix** from the zoning district code (last character: A, B, D, or X)
2. **Determine street width** — wide street (≥75 ft) or narrow street (<75 ft). If the lot faces both, use the wider street's rules for that frontage
3. **Apply base height range** — the building's streetwall must be between the minimum and maximum base heights
4. **Apply setback** — above the base height, the building must set back the required distance
5. **Apply max height** — the building cannot exceed the maximum height, period (unlike height-factor districts)
6. **Check lot coverage** — interior lots vs corner lots have different maximums

## Qualifying Ground Floor

In contextual districts, the ground floor of street-facing facades has additional requirements:
- Minimum 65% of the ground floor street wall must be transparent (windows/doors)
- Maximum 2 ft of blank wall below the window sill
- Ground floor ceiling height minimum: 13 ft (commercial) or 10 ft (residential)
- No blank walls longer than 25 ft without fenestration

## Tower-on-Base (B suffix)

R7B and R8B districts allow a tower-on-base form:
- The base must fill the streetwall to the required base height
- Above the base, a tower portion may rise to the maximum height
- The tower portion has a maximum footprint (typically 40% of lot area)
- The tower must be set back from the base
