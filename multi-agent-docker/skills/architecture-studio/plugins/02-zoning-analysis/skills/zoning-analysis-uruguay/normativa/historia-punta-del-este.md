# Historical Zoning — Punta del Este Peninsula

## Regulatory Timeline

### Pre-1959
Basic regulations. Tall buildings authorized individually by the Junta Departamental as exceptions.

### 1959 Normativa
Allowed "edificios pantalla" (screen buildings) that blocked views. Created the conditions that later codes tried to correct.

### 1974 — Decreto 3289 (Ordenanza General de Construcciones)
The first comprehensive building code for Maldonado department.

**Key provisions for the peninsula:**
- Articles 69 and 82 regulated building heights
- Towers allowed only for lots ≥ 1/4 manzana, on Gorlero and its two parallel streets
- Volumetric rules: fixed heights, low occupation, slenderness requirements
- Designed to prevent "edificios pantalla" from the 1959 code
- **No zone-based use restrictions** — the peninsula was treated as one area differentiated by location (Gorlero vs. rest), not by permitted use type
- **No single-family-only restriction** in the Faro area — that came in 1997

Described in the press as "un auténtico plan urbanístico" developed between Intendencia technicians, Ministerio de Turismo, Ministerio de Obras Públicas, and professional associations (SAU, AIU).

**Full text not digitized.** Would need to come from Maldonado municipal archives or Junta Departamental historical records.

### 1977 — Decreto 3347 (Anexo I of the OGC)
Responding to the construction boom, the Intendencia suspended all height construction on the peninsula:

- **Height suspended** on the entire peninsula except La Pastora
- La Pastora: max 30m, min lot 3,500 m² with 45m on one side
- Originally "temporary" pending a "Plan Director" — became permanent
- **Suspended HEIGHT, not USE** — no new use restrictions added
- Rationale: infrastructure (water, sewer, electricity, transport) couldn't support the building volume

Source: Junta de Vecinos expediente 15.890 (1977), suspended articles 69 and 82 of the OGC.

### 1997 — Decreto 3718 (Current TONE)
Replaced the entire 1974 OGC. Key changes for the peninsula:

- **Created the zone system**: 1.1 Faro, 1.2 Centro, 1.3 La Pastora
- **Introduced use restrictions by zone** — Faro residential subzone (1.1.2) restricted to single-family only
- New parametric system: FOS, FOT, FOS Verde
- Zone-specific setbacks and heights
- Formalized the height restrictions that had been in place since 1977

## Non-Conforming Use (Grandfathered)

Buildings constructed under the 1974 code with uses now prohibited under the 1997 code (e.g., commercial/office in Faro residential zone) operate as **legal non-conforming uses**:

- The existing use can continue
- Expansion of the non-conforming use is typically restricted
- If substantially destroyed, rebuilding the non-conforming use may not be permitted
- Change of use within the same category may trigger review

This is relevant for properties in the Faro zone (1.1.2) that predate 1997 and operate as commercial, office, hotel, or multi-family residential.
