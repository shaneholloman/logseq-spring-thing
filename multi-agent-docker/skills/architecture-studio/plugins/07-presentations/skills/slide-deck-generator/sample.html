Heading + Body

This is a page for explaining an idea in a paragraph

Use this slide type whenever you need to pair a headline with a supporting paragraph. The heading states the insight, the body provides context. Keep the paragraph to 2–3 lines — if you need more, split into two slides.

ALPA
Heading + List

This is a page for bullet points

  • Each bullet states one discrete point or finding
  • Keep to 3–6 bullets — more than that needs a second slide
  • Lead with the action or insight, not background
  • Works well on white or grey backgrounds
  • Alternate with body-text slides for rhythm
