# Code Citations

These conventions apply whenever referencing building codes, standards, or regulations.

## Format

- Always cite the **edition year**: "IBC 2021 §1004.5", not "IBC §1004.5" or "the building code"
- Use the **section symbol** (§) followed by the section number: `IBC 2021 §903.2.1`
- For tables: `IBC 2021 Table 1004.5` (not "Table 5" or "the occupancy table")
- For ASTM/ISO standards: include the full designation and year: `ASTM E119-20`, `ISO 14025:2006`

## Common Codes — Use These Abbreviations

| Code | Full Name |
|------|-----------|
| IBC | International Building Code |
| IRC | International Residential Code |
| IFC | International Fire Code |
| IMC | International Mechanical Code |
| IPC | International Plumbing Code |
| IECC | International Energy Conservation Code |
| ADA | Americans with Disabilities Act Standards for Accessible Design |
| NFPA | National Fire Protection Association (cite specific standard: NFPA 13, NFPA 72) |
| ASHRAE | American Society of Heating, Refrigerating and Air-Conditioning Engineers (cite specific standard: ASHRAE 90.1-2019) |

## Jurisdiction Awareness

- Always note that codes are **adopted and amended locally**: "IBC 2021 §1004.5 (check local amendments)"
- When a skill targets a specific jurisdiction (e.g., NYC), cite the local code: "NYC Building Code §1004.5" or "NYC Zoning Resolution §23-00"
- Never state "this complies with code" — instead say "this appears consistent with [specific code section]"
- Flag when requirements **vary by jurisdiction** or when the applicable edition is uncertain
